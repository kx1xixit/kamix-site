Version 1.1.2-early-access

March 1, 2026 Kane
changelog

This update introduces a dedicated checks service and a pre-boot environment for VFS clearing, which now triggers a system reboot. Additionally, it streamlines the system by removing demo sprites and phasing out vfs wipe and configuration commands in favor of standard VFS operations.

Adds:

  • checks, the service & VFS checking service.
  • A new pre-boot environment specifically for VFS clearing.

Changes:

  • Removed the built-in “What can Kamix do?” demo sprites if they existed.
  • vfs clear now reboots Kamix to a pre-boot environment.

Removes:

  • vfs wipe as it is now unsafe.
  • kx set-config as it was superseded by vfs write.
  • kx list-config as it was superseded by vfs list.
